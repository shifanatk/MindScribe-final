# MindScribe Icons

Create these icon files in `src/main/resources/images/`:

## Icon Requirements:
- **home-icon.png** - Simple house/home icon
- **journal-icon.png** - Book or notebook icon  
- **analytics-icon.png** - Chart or graph icon
- **ai-icon.png** - Brain or AI icon
- **settings-icon.png** - Gear icon
- **profile-icon.png** - User/person icon

## Recommended Size: 24x24px
## Format: PNG with transparency
## Style: Simple line icons matching the ethereal theme

## Sample SVG to PNG conversion:
You can create these using:
- Material Design Icons
- Font Awesome icons  
- Simple line drawings in any graphics tool

## Color: #374151 (Charcoal Mist)
These will be styled via CSS, so the base color doesn't matter as much.
